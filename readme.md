# Donneur.ca

## A complete solution for homeless to receive cashless donations fast and securely. 

This project is a full all-in-one system to help homeless people get back on their feet. It allows shelters, pedestrians and homeless to all connect in order to help lives.


## Features

* Cashless donation solution
* App for shelters to display their occupancies and announcements
* App for homeless that displays their earnings, shelters occupancies and more support
* App for pedestrians wanting to donate or participate in helping people in situation of homelessness

## Technologies

- React.js
- React Native
- Tailwind
- Python
- SQL
- Typescript
